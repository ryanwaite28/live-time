Categories/Location

Categories are a way for users to find artists, venues and events! To properly enter categories, just make a comma-separated list! For example:

hip-hip, rap, reggae, soul, etc...

It cannot be more than 250 characters.

Same thing with location, you can make it a comma-separated list. For example:

Los Angeles, CA, California, etc...

Others can also find you by location!

Time Input

The hours input uses the 24-hour format for international use. The allowed values are from 0-23. Basically any number above 12, subtract by 12 to get the hour. For example:

00 = 12:00 AM, 04 = 4:00 AM, 11 = 11:00 AM, 13 = 1:00 PM, 15 = 3:00 PM, etc...

Here is a link that better explains/depicts this format: Military Time
